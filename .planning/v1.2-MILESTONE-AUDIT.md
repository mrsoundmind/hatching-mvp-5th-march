## Tech Debt (Non-Blocking)

1. `findUserByCustomerId` in `webhookHandler.ts` uses raw SQL instead of `IStorage` — if `STORAGE_MODE=memory`, this will hit the DB pool directly
2. Stripe SDK v20+ types missing `current_period_end` on Subscription — uses `(subscription as any)` cast
3. AccountPage monthly chart shows aggregate only (no day-by-day breakdown endpoint)
4. Conversation compaction is feature-flagged off by default
5. Reasoning cache is ephemeral (in-memory, lost on server restart)
