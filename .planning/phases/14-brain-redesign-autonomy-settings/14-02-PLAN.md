Upload endpoint (Plan 01):
```
POST /api/projects/:id/brain/upload
Content-Type: multipart/form-data
Field: "document" (single file)
Returns: updated Project object (201)
Errors: 400 (invalid type/no file), 413 (over 10MB), 404 (not owned)
```

Delete endpoint (Plan 01):
```
DELETE /api/projects/:id/brain/documents/:docId
Returns: updated Project object (200)
Errors: 404 (doc not found or not owned)
```

PATCH endpoint (existing):
```
PATCH /api/projects/:id
Body: { executionRules: { autonomyEnabled, autonomyLevel, inactivityTriggerMinutes, ... } }
```

```typescript
// brain.documents[].type union (extended)

// executionRules (extended)
{
  autonomyEnabled?: boolean;
  autonomyPaused?: boolean;
  inactivityAutonomyEnabled?: boolean;
  autonomyLevel?: 'observe' | 'propose' | 'confirm' | 'autonomous';
```
